# Farní Dar

Webová aplikace pro digitální sbírky ve farnostech.

## Funkce
- Vyhledávání kostelů podle polohy
- Interaktivní mapa
- Darování (prototyp – simulovaná platba)
- Nedělní liturgické texty

## Spuštění lokálně
```
npm install
npm run dev
```

## Nasazení
Projekt je připravený pro nasazení na Vercel – stačí propojit s GitHub repozitářem.
